3. API онлайн-библиотеки

Основной функционал:

• CRUD для книг, авторов, жанров
• Книга может иметь одного или нескольких авторов
• Поиск по названию книги и имени автора
• Фильтрация по жанру

Дополнительный функционал (опционально):

• Рейтинг книги (пользователь может оценить)
• Добавление книг в "Избранное"
• Возможность загрузки обложки
• История просмотров или чтения
